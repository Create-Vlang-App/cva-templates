# Contributing

Thanks for improving the Create Vlang App template bank.

1. Open or pick a GitHub issue.
2. Branch from `main`: `feat/<issue>-<slug>`.
3. Follow [docs/AUTHORING.md](docs/AUTHORING.md) for template and extension layout.
4. Run `python scripts/ci/validate-registry.py` and `v test .` inside changed templates.
5. Open a PR with `Closes #N`.

Architecture and merge semantics: [docs/ARCHITECTURE.md](docs/ARCHITECTURE.md).
