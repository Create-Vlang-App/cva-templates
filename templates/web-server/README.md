# web-server

Minimal [veb](https://github.com/vlang/v/tree/master/vlib/veb) HTTP server for Create Vlang App.

## Run

```bash
v run .
```

Visit `http://127.0.0.1:8080/` and `/health`.

## Test

```bash
v test .
```

## Docs

See [docs/README.md](docs/README.md) for structure, API, configuration, testing, and deployment.

## Quality

See [QUALITY.md](QUALITY.md) (M1 bar). Compatible addons: `github-setup`, `v-docker`, `v-postgres`, `v-sqlite`, `development-container`, `v-fmt-vet`.
